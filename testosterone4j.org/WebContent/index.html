Testosterone4J

System Test Automation. Easy and Maintainable.

Model your UI

Model the User Interface of your application using Testosterone4J's DSL. You only need to include the information relevant for test in your model.

Script your tests

Script your tests based on the modeled UI. Testosteron4J assists you with code-completion, templates and quick-fixes and will offer you only test-steps possible at the current state of your UI.

Generate and Run

Testosterone4J will generate test data for you based on the information in your UI-Model and calculate the number of test cases needed for the specified coverage.

INSTALLATION INSTRUCTIONS

01
INSTALL ECLIPSE KEPLER

Testosterone4J is implemented based on Eclipse so you need to have an Eclipse IDE (Kepler 4.3) and Java Runtime Environment installed. You can download Eclipse here

02
INSTALL TESTOSTERONE4J

Install Testosterone4J from the Update Site: http://testosterone4j.org/updates/latest using Help->Install new software... in Eclipse.
